# kartochki.online

Frontend skeleton for a SEO-first SaaS that helps marketplace sellers generate product-card visuals.

## Stack

- Next.js 16
- React 19
- TypeScript
- SCSS Modules
- Jotai

## Project Shape

- `src/app` for Next.js routing only
- `src/views` for route-level compositions
- `src/widgets` for page sections
- `src/features` for user scenarios
- `src/entities` for domain models
- `src/shared` for shared infrastructure

See `docs/architecture.md` for the full rationale.

## Development

```bash
npm install
npm run dev
```
